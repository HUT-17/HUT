3.2.1

UD19142B

8th Apr. 2020

Fixed bugs.

3.2.0

UD17897B

20th Dec. 2019

  • Optimized the user manual.

    • Restructured the document and changed the user interface style of the document.

    • Added more details for feature descriptions. For details, see Common Features, Trigger, and Image Processing Features.

    • Added video in Configure LUT.

    • Added sample use case in GigE Vision Action Command.

  • Supports new file formats for the captured pictures and recorded video files. For details, see Capture and Recording.

  • Supports enable the PC to respond to the keyboard shortcut request from the Software in priority if a same keyboard shortcut of another software or program exists. For details, see Shortcut.

  • Supports fast accessing the folder containing the SDK development documents and demos. For details, see Help.

  • Supports sticking a GigE Vision camera to the top and its GVCP configuration. For details, see Other Features.

  • Supports sticking a USB3 Vision camera to the top. For details, see Other Features.

  • Supports GenTL, which allows you to connect CoaXPress cameras . For details, see GenTL Management.

  • Supports use Event Monitor to monitor USB3 Vision cameras.

  • Updated the UI for configuring white balance under Bayer format. For details, see Configure White Balance (Bayer).

  • Adjusted the procedures for drawing ROI and AOI. For details, see Draw ROI.

  • Supports displaying feature difference when importing features to a single camera. For details see Import/Export Features of a Single Camera.

  • Supports opening local images and video files. For details, see View Local Image and Video.

  • Supports more configurations for the cross line. For details, see Set Cross Line.

  • Supports setting grids on images. For details, see Configure Grid.

  • Supports using histogram for evaluating image quality. For details, see View Histogram.

  • Supports selecting the information item(s) which need to be displayed on IP Configurator. For details see IP Configurator.

  • Supports selecting the information item(s) which need to be displayed on Firmware Updater. For details, see Firmware Updater.

  • Supports selecting information item(s) which need to be displayed on Bandwidth Manager and connecting cameras on Bandwidth Manager. For details, see Bandwidth Manager.

  • Added mutual exclusion feature for the Periodically Send parameter and Request Acknowledge parameter of Action Command. For details, see GigE Vision Action Command.

  • Supports selecting information item(s) which need to be displayed on Embedded Information window. For details, see View Embedded Information.

  • Supports batch settings in NIC Configurator. For details, see NIC Configurator

  • Supports manually refreshing logs on Log Viewer, viewing process name and process ID in the log information, and adding DLL(s) to display their logs. For details, see SDK Logs.

3.1.0

UD12091B

11th Oct. 2018

  • Added the Add Camera Link Camera chapter to introduce how to add Camera Link cameras to the client and the camera status.

  • Added the Bandwidth Manager chapter to introduce how to optimize bandwidth of the connected cameras via Bandwidth Manager.

  • Added the Environment Diagnostics chapter to introduce how to diagnose and optimize the running environment and the secondary development environment.

  • Added the GigE Vision Action Command chapter to introduce GigE Vision action command, which can be used to trigger actions in multiple cameras in a network simultaneously.

  • Added the Keyboard Shortcut Settings chapter to introduce how to configure keyboard shortcuts for some frequently-used operations.

  • Edited the Configure Image Processing Features to introduce changes in the operations of drawing ROI.

  • Edited the Log Viewer chapter to introduce how to configure SDK log service.

3.0.0

UD10830B

6th Jul. 2018

  • Restructured the whole user manual due to UI changes.

  • Added the Log Viewer chapter to introduce the tool in which you can view and filter logs from different DLLs.

  • Added the View live View in Full Screen chapter to introduce how to view live view in full screen mode when the window division is set to 1-window mode and multiple-window mode.

  • Added Window Division to introduce how to set window division and customize window division.

  • Added the View Embedded Information chapter to introduce how to view the embedded image information of multiple cameras simultaneously.

2.4.0

UD09133B

6th Feb. 2018

  • Updated the Interface Introduction chapter to describe updates in the menu (added buttons and moved camera feature settings to Camera > Basic Features).

  • Added the Import/Export Camera Feature via File Access chapter to describe how to import and export camera features via the tool called File Access, which can import and export different files.

  • Added the Batch Import and Export Features chapter to describe how to import and export features for multiple cameras.

  • Added the Event Monitor chapter to describe the tool called Event Monitor, on which the messaging channel events can be displayed to help you find the causes of issues that may occur during the use of your camera.

  • Added the Multicast Configuration chapter to describe how to set multicast.

  • Updated the Configure Camera IP Address chapter to describe how to configure IP addresses of multiple cameras simultaneously.

  • Updated the Update Camera Firmware via Firmware Updater chapter to describe how to update firmwares of multiple cameras simultaneously.

2.3.1

UD08248B

25th Nov. 2017

Updated the View Status chapter to show real time bandwidth, errors, packet lost in Status page is supported.

2.3.0

UD06966B

29th Aug. 2017

  • Restructured the whole manual to make the manual more user-friendly.

  • Updated the Running Environment chapter to update the PC system requirements.

  • Added the Packet Resending Settings chapter to describe the packet resending settings.

  • Added the Live View chapter to describe live view for single camera and live view for multiple cameras, as well as other operations during live view, such as configuring cross line to help adjusting the position of the object in the live view image.
